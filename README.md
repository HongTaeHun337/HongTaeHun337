    ## 💻 Projects

    ### 1. TripTo (여행 동행 매칭 및 커뮤니케이션 플랫폼)
    **프로젝트 개요:** 혼자 여행하는 사람들을 위해 취향과 일정이 맞는 동행자를 찾아주고, 안전하게 소통할 수 있도록
  지원하는 종합 여행 커뮤니티 웹 서비스입니다.
    **사용 기술:**
    * **Backend:** Java 11, Spring Framework (5.3.9), Spring Security, MyBatis, WebSocket, JavaMail
    * **Frontend:** JSP, HTML/CSS, JavaScript
    * **Database & Storage:** Oracle DB, HikariCP, Cloudinary (이미지 호스팅)

    **💡 전반적인 주요 기능 (Team)**
    * **보안 및 인증:** Spring Security를 활용한 인증/인가, JavaMail 연동 이메일 인증 회원가입
    * **커뮤니티:** 다중 이미지 업로드가 가능한 여행 기록(Travel Post) 게시판 및 댓글 시스템
    * **부가 기능:** 관리자 대시보드(Admin), 불량 사용자 신고 시스템, 맞춤형 게시물 추천

    **🔥 나의 주요 담당 역할 및 기여도 (My Contribution)**
    * **사용자 맞춤형 매칭 시스템 구현:**
      * 사용자들의 일정, 여행지, 성향 등의 조건을 비교하여 적합한 동행자를 찾아주는 매칭 로직 설계 및 UI 구현
      * MyBatis를 활용하여 다양한 필터링 조건에 대응하는 동적 SQL 쿼리 작성
    * **WebSocket 기반 실시간 채팅 서비스 구축:**
      * 매칭된 사용자 간의 원활한 소통을 위해 `spring-websocket`을 활용한 실시간 채팅 채널 구현
      * 채팅 세션 관리 및 송수신된 메시지의 데이터베이스 저장/불러오기 기능 구현

    ### 2. 씨네21 DB 모델링 (Cine21 Database Modeling)
    **프로젝트 개요:** 국내 최대 영화 포털 '씨네21'의 복잡한 데이터 구조를 벤치마킹하여, 대용량 트래픽 확장이 가능한
  형태의 영화 정보 관리 RDBMS 모델링을 수행한 프로젝트입니다.
    **사용 기술:** Oracle DB, eXERD, SQL Developer, SQL(DDL, DML)

    **💡 전반적인 주요 기능 (Team)**
    * **대규모 데이터베이스 아키텍처 설계:** 총 18개의 핵심 테이블과 20개의 시퀀스로 구성된 실무 수준의 복잡도 설계
    * **메타데이터 통합 관리:** 영화 기본 정보뿐만 아니라 감독, 배우, 스태프, 시놉시스, 리뷰 등 방대한 영화 관련
  메타데이터 통합 구조 구축

    **🔥 나의 주요 담당 역할 및 기여도 (My Contribution)**
    * **정규화 및 다대다(N:M) 관계 최적화:**
      * 영화-배우, 영화-장르 간의 복잡한 다대다 관계를 해소하기 위해 논리적 매핑 테이블을 도출하여 물리 모델에 반영
    * **사용자 경험을 고려한 리뷰 구조 설계:**
      * 일반 관람객의 별점 시스템과 전문가의 심층 비평 데이터 구조를 분리 설계하여 데이터의 무결성 및 확장성 확보
    * **제약조건 설계 및 실무 더미 데이터 구축 (DML 작성):**
      * 국가, 영화, 장르 등 실제 개봉작 20여 편 기반의 더미 데이터를 구축하여 시스템 검증
      * 시퀀스(`NEXTVAL`)와 외래키(`FOREIGN KEY`) 제약 조건을 완벽하게 연동하여 데이터 무결성 보장
